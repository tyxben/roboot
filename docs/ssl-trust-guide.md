# SSL 证书信任指南

自签名证书需要手动在手机上信任才能使用 HTTPS。

## 📱 iOS (iPhone/iPad)

### 方法 1：通过 Safari 直接信任（推荐）

1. **首次访问 HTTPS 地址**
   ```
   https://192.168.1.27:8765
   ```

2. **看到安全警告**
   - 点击 "显示详细信息"
   - 点击 "访问此网站"
   - 再次点击 "访问此网站" 确认

3. **完成！**
   - 证书已临时信任
   - 可以正常使用摄像头和语音

### 方法 2：永久信任证书（可选）

1. **下载证书**
   ```
   https://192.168.1.27:8765/static/cert.pem
   ```

2. **安装描述文件**
   - 设置 → 已下载描述文件
   - 点击安装
   - 输入密码确认

3. **信任根证书**
   - 设置 → 通用 → 关于本机
   - 证书信任设置
   - 启用对该证书的完全信任

## 🤖 Android

### 方法 1：Chrome 直接信任（推荐）

1. **首次访问 HTTPS 地址**
   ```
   https://192.168.1.27:8765
   ```

2. **看到隐私错误**
   - 点击 "高级"
   - 点击 "继续前往 192.168.1.27（不安全）"

3. **完成！**
   - 可以正常使用

### 方法 2：系统级信任（可选）

1. **下载证书**
   ```
   https://192.168.1.27:8765/static/cert.pem
   ```

2. **安装证书**
   - 设置 → 安全 → 加密与凭据
   - 安装证书 → CA 证书
   - 选择下载的 cert.pem
   - 命名为 "Roboot"

3. **重启 Chrome**

## ❓ 常见问题

### Q: 为什么需要信任证书？
A: 自签名证书不被浏览器默认信任。这是安全机制，防止中间人攻击。

### Q: 安全吗？
A: 
- ✅ 在家里 WiFi 使用是安全的
- ✅ 流量已加密，无法被窃听
- ⚠️ 不要在公共 WiFi 使用
- ⚠️ 不要在不信任的网络使用

### Q: 每次都要信任吗？
A: 
- iOS Safari: 每次会话需要确认
- Android Chrome: 只需要第一次确认
- 永久信任: 安装证书到系统（方法 2）

### Q: 可以不用 HTTPS 吗？
A: 
- 浏览器安全策略要求 HTTPS 才能访问摄像头和麦克风
- 没有 HTTPS，语音和摄像头功能无法使用
- 只用文字聊天可以不用 HTTPS
